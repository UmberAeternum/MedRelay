# Demo script

Open `/medrelay` and point out “Demo — synthetic data only.” The public path is a zero-cost deterministic safety engine: it makes no external AI call and shows the exact offline label. Choose a labelled respiratory, digestive, urinary, skin, neurological, injury, Telugu, Hindi, English, or mixed-language scenario. Show one focused question at a time, then create the editable clinician-review handoff, edit a field, expand Evidence and audit trail, and copy it. Add “Actually…” as a correction and regenerate to show the latest correction superseding the displayed symptom while the transcript evidence remains. Finish with Reset demo.

The optional OpenAI Responses API integration is server-only, opt-in, and not enabled or claimed for this demo. Do not describe a deterministic result as GPT output or imply a clinician has reviewed the draft.
