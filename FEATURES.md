# Features

## Implemented

- Public synthetic demo without OAuth or MySQL
- Telugu, Hindi, English, and mixed-language input
- Server-controlled expiring sessions with reset and ownership token
- Patient-only deterministic warning-sign and negation screening
- Pre-provider emergency bypass and public rate limiting
- Injection screening across trusted patient history
- GPT-5.6 Responses API adapter with `store: false`, HMAC safety identifier, and Zod Structured Outputs
- Semantic/cross-field validation and urgency reconciliation
- Per-field evidence IDs, quotes, kinds, confirmation labels, and transcript audit
- Editable/copyable handoff and typed fallback when voice input is unavailable

## Not implemented or claimed

- Diagnosis, probabilities, prescriptions, doses, treatment changes
- Automatic appointments, emergency dispatch, or completed clinician review
- Durable multi-instance demo storage or regulatory/clinical certification
